# aminome

**A**dd all **mi**sskey **no**tes to **Me**ilisearch.

## Target version

- Misskey v13.12.1
- Meilisearch v1.1.1

## How to use

Install python package.

```sh
pip3 install -r requirements.txt
```

Configure and run.

```sh
python3 ./aminome.py
```
